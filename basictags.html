HTML Basic Tags

1. <!DOCTYPE html>

Defines the document type (not visible on the page).

2. <html> ... </html>

This tag wraps the entire HTML document. You're viewing content inside <html> now.

3. <head> ... </head>

Content like <title> and metadata are inside <head>. It's not visible on the page.

6. <base>

Defines a base URL for relative links (not directly visible here).

7. <link>

Used to include external CSS (like stylesheets). Not shown directly in the browser.
