HTML Frames, Audio, Video & Others

1. <iframe> ... </iframe>

Used to embed another webpage within the current page.

2. <audio> ... </audio>

Used to embed sound content.

3. <video> ... </video>

Used to embed video content.

4. <source />

Specifies multiple media resources for <audio> or <video>.

5. <track />

Used to specify text tracks for <audio> or <video> (like subtitles).

6. <embed />

Embeds external applications like PDFs or flash.

7. <object> ... </object>

Defines an embedded object (like multimedia or plugin).

8. <param>

Defines parameters for plugins embedded using <object>.
