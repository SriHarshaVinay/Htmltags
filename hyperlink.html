HTML Links

<a>...</a>

Defines a hyperlink, used to link from one page to another.

Example:

<a href="https://www.example.com">Visit Example.com</a>

Output:

Visit Example.com

Use the href attribute to specify the link destination. You can also use target="_blank" to open the link in a new tab.
