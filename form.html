HTML Form Tags

1. <form> ... </form>

Creates an HTML form for user input.

2. <input />

Defines an input field.

3. <label> ... </label>

Defines a label for an input element.

4. <button> ... </button>

Creates a clickable button.

5. <textarea> ... </textarea>

Multiline text input.

6. <select> ... </select>

Dropdown list.

7. <option> ... </option>

Defines items in a dropdown list.

8. <optgroup> ... </optgroup>

Groups related options in a dropdown.

9. <fieldset> ... </fieldset>

Groups related form fields.

10. <legend> ... </legend>

Defines a caption for the <fieldset>.

11. <datalist> ... </datalist>

12. <output> ... </output>
