HTML Images

<img />

Displays an image.



<map> and <area />

Defines an image map with clickable areas.



<picture>

Used for multiple image sources depending on screen or device.



<figure> and <figcaption>

Used to group media content and a caption.


<canvas>

Used for drawing graphics via JavaScript.



<svg>

Defines SVG (Scalable Vector Graphics) content.
